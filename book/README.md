# Physical AI & Humanoid Robotics Book

This book is built using [Docusaurus](https://docusaurus.io/), a modern static website generator.
For Docusaurus documentation, please refer to: [https://docusaurus.io/docs](https://docusaurus.io/docs)

## Installation

Navigate to the `book/` directory and install dependencies:

```bash
cd book
npm install
```

## Local Development

To start a local development server and open a browser window:

```bash
cd book
npm start
```

Most changes are reflected live without having to restart the server.

## Build

To generate static content into the `build` directory for production:

```bash
cd book
npm run build
```

The static content will be generated in `book/build`.

## Deployment to GitHub Pages

The deployment is handled via GitHub Actions. Pushing to the `main` (or `master`) branch will trigger the workflow to build and deploy the website to GitHub Pages.
